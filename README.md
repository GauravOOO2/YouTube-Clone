## Run Locally

Clone the project in your machine

```bash
  git clone https://github.com/GauravOOO2/YouTube-Clone.git
```

Go to the project directory

```bash
  cd YouTube-Clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
